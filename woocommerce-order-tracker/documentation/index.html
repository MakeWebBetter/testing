OVERVIEW

Woocommerce Order Tracker allow the customer to track their orders on the site in an user interactive manner.

FEATURES

The extension provides various features :

  • Shop admin can enable/disable this tracking feature.

  • Shop admin can enable the use of the custom order status provided.

  • Shop Manager can add an estimated delivery date and time for the orders.

  • Order tracking is divided in different sections.

  • Shop admin can select order statuses to be displayed in different sections while tracking.

  • Shop admin can create custom order statuses for making more interactive frontend for tracking order.

  • A graphical and user interactive frontend for tracking order.

  • Seperate tracking form for not logged in Customers.

  • Easy Customisation.

  • Simple and Easy to use.

 

INSTALLATION

AUTOMATIC INSTALLATION

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser.

Steps are as follows :

1. Download the zip folder of the plugin Woocommerce Order Tracker.

2. Once dowloaded install plugin through browsing and activate it.

MANUAL INSTALLATION

Manual installation of plugin is another option to install plugin to your wordpress-environment. The manual installation method involves downloading our Woocommerce Order Tracker Extension and uploading it to your webserver via your favourite FTP application.

Steps are as follows :

1. Upload the Woocommerce Order Tracker folder to the /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress

WORK FLOW

ADMIN PANEL

Enable the Track Order setting from the admin panel.

Admin can select the use of the Custom status if desired. On selecting the use of Custom Order Statuses , a dropdown with new Order Statuses is shown for selecting the Order Statuses ( Order Packed, Order Dispatched, Order Shipped ).

Shop Admin needs to select the order status for the different sections of tracking as shown in the image. The sections are APPROVAL, PROCESSING and SHIPPING.

The Order Status Selected for different sections will be listed accordingly while order tracking.

In the Common Settings Tab, the shop admin can fill in the messages to be displayed with the order statuses during order tracking.

Once an order is placed Shop Admin can provide an estimated date and time for order delivery from the order edit page.

                                                

FRONTEND

Once an order is placed,

FOR LOGGED IN USERS --

Customer can track thier order form the My Account -> Order section .

FOR NOT LOGGED IN USERS --

Not Logged in Users can also track their order from the TRACK YOUR ORDER page by just providing the ORDER ID and the customer EMAIL ID.

Just an click on Track Order and Customer will see the entire details and processing of their ORDERS.

The order statuses through which the order is passed are shown in GREEN CIRCLE and the current status of the order is shown with DOUBLE CIRCLE. The message with all completed statuses ae shown on hovering over the circle.
